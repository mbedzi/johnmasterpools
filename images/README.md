## How to Replace with Your Own Images

### Option 1: Use Your Own Photos
1. Take photos of your actual pool work
2. Save them in this `images/` folder
3. Update the HTML file to reference your local images

### Option 2: Find More Free Pool Images
- **Unsplash**: https://unsplash.com/s/photos/swimming-pool
- **Pexels**: https://www.pexels.com/search/swimming%20pool/
- **Pixabay**: https://pixabay.com/images/search/swimming%20pool/

### Recommended Image Specifications:
- **Size**: 1000x600 pixels (or similar ratio)
- **Format**: JPG or WebP
- **Quality**: High resolution for web use
- **File size**: Under 500KB each for fast loading

## Steps to Add Your Own Images:

1. **Save your images** in this folder with descriptive names:
   - `pool-repair-1.jpg`
   - `pool-cleaning-1.jpg`
   - `pool-equipment-1.jpg`
   - `pool-clear-1.jpg`

2. **Update the HTML** in `index.html`:
   ```html
   <!-- Replace the Unsplash URLs with your local images -->
   <img src="images/pool-repair-1.jpg" alt="Pool tile and coping repair work" class="gallery-image">
   ```

3. **Optimize images** for web:
   - Use tools like TinyPNG or Squoosh to compress
   - Ensure images load quickly on mobile devices
